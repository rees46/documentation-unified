# Курсы и образовательные программы

Серия бесплатных курсов по CRM-маркетингу от REES46.

## Категория «Старт»

Идеально для новичков, кто хочет структурировать разрозненные знания в связную систему.

1. [Системный курс по CRM-маркетингу](https://stepik.org/course/234225)
2. [REES46: CRM-маркетинг полного цикла для роста ценности клиента](https://stepik.org/course/254317)
3. [CRM-сегментация: объединяя стратегию и тактику для роста бизнеса](https://stepik.org/course/250965)

## Категория «Рост»

Для тех, кто освоил основы, чтобы уверенно выстраивать актуальную CRM-стратегию на практике.

1. [CRM-старт: как заложить основу успешного года в январе](https://stepik.org/course/261231)
2. [CRM-эксперимент: запускаем рост в феврале](https://stepik.org/course/268184)
3. [CRM-рывок: как усилить рост показателей в марте](https://stepik.org/272383)
4. [CRM-масштаб: как развить успех в апреле](https://stepik.org/279219)
5. [CRM-удержание: как сохранить клиентскую базу в мае](https://stepik.org/284169)
6. [CRM-легкость: как не усложнять коммуникацию в июне](https://stepik.org/288172)

## Категория «Эксперт»

Для тех, кто хочет выйти за рамки CRM-маркетинга, используя потенциал других дисциплин.

1. [CRM-маркетинг в эпоху поколенческого разрыва](https://stepik.org/course/247096)
2. [Кросс-дисциплинарный CRM-маркетинг: теория и практика](https://stepik.org/251119)
2. [Мономиф лояльности. CRM-маркетинг по канонам "Пути Героя"](https://stepik.org/253175 )
