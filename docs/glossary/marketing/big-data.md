# Big Data

Big Data (большие данные) — огромные массивы данных из социальных сетей, онлайн-поисковиков, клиентских баз и многих других источников информации. Объём таких данных растёт с большой скоростью, поэтому обработать их традиционными способами невозможно. Для работы с Big Data разрабатывают специальные инструменты и алгоритмы.
