# Автоматическое удаление неактивных анонимных профилей

Ежедневно платформа автоматические удаляет профили анонимных неактивных пользователей. 

Критерии удаления профиля:

- Отсутствие контактных данных
- Отсутствие активности в течение года

Также удаляются брошенные корзины, если пользователь не был активен в течение трёх месяцев.

## Отображение в отчётах

Автоматическое удаление профилей можно отследить в статистике раздела "Динамика аудитории".

Показатель **Всего** может показывать динамику, которая может быть интерпретирована как массовое удаление профилей.

Наиболее вероятная причина такой динамики: автоматическое удаление неактивных анонимных профилей.
