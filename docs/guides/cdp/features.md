# Сравнение редакций CDP

## CDP

- Дополнительные виды событий
- Дополнительные свойства пользователей:
  - Булевые
  - Строковые
  - Даты
  - Списки
  - Массивы
  - JSON
- Нишевые расчетные свойства: пол, возраст, размер одежды, размер обуви, домашние животные, предпочтения в ювелирных изделиях, дети (пол, возраст, количество)
- Событие "Скоро день рождения"

## Сегментация

- Сегментация по дополнительным свойствам пользователя
- Сегментация по дополнительным событиям
- Расширенные клиентские метрики
- Системные события перемещения клиентов между сегментами с возможностью привязки триггерных рассылок
- Сегментация по системным событиям
- Сегментация по расчетным свойствам профиля

## RFM

- Расширенные клиентские метрики
- Системные события перемещения клиентов между сегментами с возможностью привязки триггерных рассылок

## Массовые рассылки

- Отправка рассылок по часовому поясу получателя
- Отправка рассылок по лучшему времени отправки для получателя

## Триггерные рассылки

- Таргетинг по дополнительным свойствам пользователя
- Таргетинг по дополнительным событиям
- Таргетинг по расчетным свойствам профиля
- Дополнительные системные события для триггерных рассылок
- Триггерные рассылки по дополнительным событиям CDP

## Попапы

- Таргетинг по дополнительным свойствам пользователя
- Таргетинг по дополнительным событиям
- Таргетинг по расчетным свойствам профиля

- Дополнительные параметры приоритета товара в локации для таргетинга товаров по городам и точкам продаж
