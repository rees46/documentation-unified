# Customer data platform

В основе REES46 лежит CDP – платформа пользовательских данных, позволяющая в режиме реального времени дедуплицировать посетителей, в том числе анонимных и аккуратно собирать и обрабатывать поведенческие данные каждого пользователя или устройства.

CDP является частью каждого инструмента REES46 и поставляется в двух видах:

1. Базовая версия. Входит в каждый инструмент и поставляется бесплатно.
2. Расширенная версия. Дает больше возможностей для управления профилем клиента и дополнительную функциональность в каждый инструмент.
