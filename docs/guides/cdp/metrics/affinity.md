# Привязанность к категориям и брендам

Метрика `affinity` позволяет определить привязанность покупателя к категориям и брендам. Привязанность рассчитывается именно к паре "категория + бренд", так как у клиента могут быть разные предпочтения к брендам разных категорий. 

Метрика рассчитывается для клиентов, которые оформили 3 и более заказов. Расчет выполняется в ночное время.

## Интерпретация метрики

Метрика представляет собой целое число. По его величие можно понять, насколько клиент привязан к паре "категория + бренд":

| Ситуация      | Значение | Интерпретация                    |
|---------------|----------|----------------------------------|
| Слабый сигнал | < 2      | Лёгкое предпочтение, мало данных |
| Умеренный     | 2–7      | Заметное предпочтение            |
| Сильный       | 7–17     | Явный фаворит                    |
| Очень сильный | 17+      | Клиент покупает почти только это |

## Использование

Метрику можно использовать в следующих кейсах:

- сегментация клиентов по привязанности к бренду или категории;
- проверка условий попапа;
- проверка условий блока товарных рекомендаций;
- проверка условий триггерной цепочки;
- экспорт сегмента в рекламные кабинеты.
