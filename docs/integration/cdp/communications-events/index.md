# Коммуникационные события

Раздел описывает события, связанные с рассылками. А именно: открытие рассылки, переход по нему, закрытие мобильного пуша и доставка сообщения.

В случае отправок email, большинство событий отслеживаются автоматически. Но мобильные уведомления требуют ручной передачи результатов взаимодействия с сообщением в CDP.
