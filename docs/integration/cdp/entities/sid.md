# Сессия `sid`

Сессия позволяет отделять разные сеансы посещения сайта посетителем (на самом деле - устройством). Началом сессии считается первое действите посетителя после минимум двухчасового отсутствия событий.

Для маркировки сессии используются параметры `sid` (актуальный) и `seance` (устаревший, оставлен для обратной совместимости и со временем будет убран).
