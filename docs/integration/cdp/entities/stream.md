# Стрим

Тег источника данных. Этот параметр устанавливается в [инициализации SDK](./../sdk/index.md). Если не указан, считается, что поток данных идет с сайта.

Можно задавать разные стримы для точек касания с клиентом: сайт, мобильные приложения, кассы, инфокиоски, чат-боты, колл-центр и пр.

Формат значения: строка латиницей. Мы рекомендуем использовать простые короткие значения типа `ios`, `android`, `pos`. 

Для данных с сайта можно не задавать значение `stream`: пустое значение автоматически ассоциируется с веб-сайтом.
