# Установка SDK и запуск сессии

Для автоматической идентификации анонимных посетителей на сайте и в мобильных приложениях необходимо установить SDK.

Запуск SDK проверяет наличие `did`. Если его нет, то создает. Инициализирует сессию на сервере REES46.

Актуальные SDK:

| Платформа                | Язык   | SDK                                  |
|--------------------------|--------|--------------------------------------|
| Сайт                     | JS     | [JS SDK](./js.md)                    |
| iOS                      | Swift  | [iOS SDK](./swift.md)                |
| Android                  | Kotlin | [Android SDK](./kotlin.md)           |
| Muliplatform ReactNative | JS     | [ReactNative SDK](./react-native.md) |
