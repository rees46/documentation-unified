# Атрибуция инструментов

Чтобы платформа могла рассчитать эффективность работы каждого инструмента, используется атрибуция действий пользователей.

За атрибуцию действий к инструментам REES46 отвечают параметры `source_type+source_id` (для рассылок) и `recommended_by+recommended_code` (для витринных инструментов).

SDK автоматически решают эту задачу, но в случае интеграции S2S вам потребуется самостоятельно управлять этими параметрами.
