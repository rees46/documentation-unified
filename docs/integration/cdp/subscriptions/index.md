# Управление подписками на рассылки

Данный раздел описывает API и SDK методы для управления подписками клиентов на разные типы рассылок и каналы.

Типы рассылок:

- массовые (bulk);
- триггерные (chain);
- транзакционные или сервисные (transactional).

Каналы:

- email;
- web_push;
- mobile_push;
- sms;
- telegram;
- whatsapp;
- max;
- wallet.
