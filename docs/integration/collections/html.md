# Подключение товарных коллекций на сайте через DIV

Самый простой способ установки товарных коллекций на сайт – размещение DIV-блоков.

JS SDK автоматически находит все DIV-блоки с определенным классом, получает из них идентификаторы коллекций, запрашивает информацию и верстку по ним и вставляет готовые товарные подборки на сайт.

Для автоматической отрисовки достаточно установить DIV-блоки на странице в формате:

```html
<div
    class="rees46-collection-products-results"
    data-collection-id="6"
    data-collection-callback="console.log"
    data-collection-error="console.log"
    data-collection-location="London"
    data-collection-email="my@email.com"
    data-collection-phone="70000000000"
    data-collection-external_id="..."
    data-collection-loyalty_id="..."
/>
```

Описание атрибутов тега:

| Параметр                    | Тип      | Обязателен? | Описание                                                                                                                                                                                            |
|-----------------------------|----------|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| class                       | String   | Да          | Должен содержать класс `rees46-collection-products-results`. Для white label кластеров `rees46` меняется на код, соответствующий white label                                                        |
| data-collection-id          | Integer  | Да          | Идентификатор товарной коллекции из личного кабинета                                                                                                                                                |
| data-collection-callback    | String   | Нет         | Имя JS функции, куда уйдет ответ сервера, если вы хотите обработать его сами. Если не указать, то JS SDK автоматически обработает ответ и сгенерирует верстку из шаблона, заданного в личном кабинете |
| data-collection-error       | String   | Нет         | Имя JS функции, куда уйдет ответ сервера в случае ошибочного запроса                                                                                                                                |
| data-collection-location    | String   | Нет         | Код текущего выбранного города покупателя, если у вас в товарном фиде поддерживаются локации. Если локаций у вас нет, то параметр не обязателен. Если есть, то обязателен                           |
| data-collection-email       | String   | Нет         | Email текущего посетителя, если есть                                                                                                                                                                |
| data-collection-phone       | String   | Нет         | Телефон текущего посетителя, если есть                                                                                                                                                              |
| data-collection-external_id | String   | Нет         | Внешний идентификатор текущего посетителя, если есть                                                                                                                                                |
| data-collection-loyalty_id  | String   | Нет         | Идентификатор участника внешней программы лояльности. Если у вас на сайте есть локации, то этот параметр обязателен                                                                                                                                                |
