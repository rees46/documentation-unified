# Товарные коллекции

Это витринный инструмент для формирования подборок товаров. Является гибридом между товарными рекомендациями и листингом товаров:

1. Позволяет делать подборки достаточно большими по количеству товаров, в отличие от товарных рекомендаций.
2. При этом позволяет задавать фильтры товаров контент-менеджером в личном кабинете платформы, в отличие от листинга товаров.
