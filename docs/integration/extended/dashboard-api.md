# Dashboard API

Dashboard API - отдельный API для работы с данными личного кабинета. Через него можно создавать и запускать рассылки, управлять цепочками, правами доступа и пр.

Dashboard API можно использовать для разработки собственного интерфейса над платформой, проинтегрировать возможности REES46 в свою собственную платформу или создавать узкоспециализированные интерфейсы: например, карточка клиента, которая подтягивается в интерфейс оператора колл центра во время входящего звонка.

Документация находится по адресу: https://app.rees46.ru/api-docs
