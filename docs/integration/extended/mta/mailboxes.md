# Сервисные почтовые адреса

Необходимо создать почтовые ящики для обработки фидбека почтовых сервисов. Рекомендуемые:

1. `bounce@bounce.mydomain.com` – для получения hard/soft bounce
2. `fbl@bounce.mydomain.com` – для приема жалоб на спам
3. `unsubscribe@bounce.mydomain.com` – для уведомлений о необходимости отписки

Эти почтовые ящики должны быть прописаны в конфигурации сервиса `bounce-worker`.
