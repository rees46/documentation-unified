# Поставка в виде on-premise

Так как система состоит из десятков различных компонентов, установка и настройка кластера силами клиента на данный момент не рассматривается. Вся настройка и мониторинг выполняются силами REES46.

В дальнейшем будет готов инсталлятор Kubernetes-кластера и тогда весь кластер можно будет установить силами клиента за несколько часов.

Подключение сайта и мобильных приложений к on-premise кластеру выполняется точно так же, как и к облачному кластеру. Разница только в доменах API и CDN.

:::info Совет
Для сокращения календарных сроков подключение сайта и приложений, подключение можно начать на облачном решении и затем переключить все на on-premise.
:::

Помимо настройки оборудования в on-premise и white-label кластерах требуется настройка почтовых серверов и прогрев IP-адресов.
