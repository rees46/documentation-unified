# Текущий сегмент посетителя в A/B-тесте

Метод получает значение сегмента, в котором находится посетитель. Этот сегмент используется в правилах товарных рекомендаций, цепочек и других инструментов со встроенным A/B-тестом.

Метод может быть полезен для проверки, по какой ветке правил триггерной цепочки пойдет посетитель.
