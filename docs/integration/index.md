# План интеграции

Интеграция любого инструмента REES46 начинается с базовой интеграции CDP:

1. [Установить SDK](./cdp/sdk.md) на сайт и мобильные приложения и старт сессии
2. Настройка [стандартных событий](./cdp/standard-events/index.md)
3. Настройка [кастомных событий](./cdp/custom-events.md)
4. Передача [данных профиля](./cdp/profile/index.md)
5. [Импорт каталога](./catalog/index.md) товаров и услуг
6. [Синхронизация статусов заказов](./cdp/orders/index.md) и импорт розничных заказов

Эти шаги позволят настроить обмен данными, необходимый для работы всех инструментов. Платформа может работать с любым устройством, у которого есть интернет. Есть SDK и есть REST API.

Затем нужно провести настройку каждого инструмента, который вы хотите использовать. Для этого читайте соответствующие разделы.

Для управления сущностями платформы есть еще Dashboard API: https://app.rees46.ru/api-docs
