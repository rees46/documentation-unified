# Структура объекта сертификата

## Пример ответа

```json 
{
    "id": 333,
    "status": "created",
    "code": "abc---certificode-...",
    "nominal": 5000,
    "balance": 3980,
    "owner_identifier": "...",
    "owner_name": "...",
    "buyer_identifier": "...",
    "created_date": "...",
    "activated_date": "...",
    "deactivated_date": "...",
    "expiration_date": "...",
    "planned_activation_date": "...",
    "batch": null
}
```

## Описание полей

| Параметр                | Описание                                                                                           |
|-------------------------|----------------------------------------------------------------------------------------------------|
| id                      | Идентификатор сертификата                                                                          |
| status                  | Статус сертификата                                                                                 |
| code                    | Уникальный код сертификата                                                                         |
| nominal                 | Номинал сертификата                                                                                |
| balance                 | Остаток средств на сертификате                                                                     |
| owner_identifier        | Идентификатор владельца сертификата                                                                |
| owner_name              | Имя владельца сертификата                                                                          |
| buyer_identifier        | Идентификатор покупателя сертификата (покупатель может отличаться от владельца)                    |
| created_date            | Дата создания                                                                                      |
| activated_date          | Дата активации                                                                                     |
| deactivated_date        | Дата деактивации                                                                                   |
| expiration_date         | Дата сгорания                                                                                      |
| planned_activation_date | Планируемая дата отложенной активации                                                              |
| batch                   | Идентификатор партии сгенерированных сертификатов, если партия генерировалась через личный кабинет |
