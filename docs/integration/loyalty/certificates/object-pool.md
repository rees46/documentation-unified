# Структура объекта пула сертификата

## Пример ответа

```json 
{
    "id": 17,
    "status": "active",
    "flexible_nominal": 6000,
    "personal": true,
    "expiration_days": 30,
    "min_order_value": 1000,
    "payable_order_share": 70,
    "exclude_products": [ { ... }, { ... }, ... ]
}
```

## Описание полей

| Параметр             | Описание                                                                       |
|----------------------|--------------------------------------------------------------------------------|
| id                   | Идентификатор пула сертификатов                                                |
| status               | Статус пула сертификатов                                                       |
| name                 | Название пула сертификатов                                                     |
| description          | Описание пула сертификатов                                                     |
| nominal              | Базовый номинал сертификата                                                    |
| flexible_nominal     | Значение гибкого номинала, если пул с гибким номиналом                         |
| max_flexible_nominal | Максимальное значение гибкого номинала, если пул с гибким номиналом            |
| personal             | Флаг, что сертификаты именные                                                  |
| code                 | Код пула                                                                       |
| expiration_days      | Срок сгорания сертификатов, если задан                                         |
| streams[]            | Список стримов, в которых может работать сертификат этого пула                 |
| location_ids[]       | Список локаций, в которых может работать сертификат этого пула                 |
| min_order_value      | Минимальная стоимость заказа, если задана                                      |
| payable_order_share  | Максимальная доля цены товара, которую можно оплатить сертификатами этого пула |
| exclude_products     | Товары и группы товаров, которые нельзя оплатить сертификатами этого пула      |
| promo                | Флаг рекламного сертификата                                                    |
| activation_delay     | Отсрочка активации сертификата в днях                                          |
