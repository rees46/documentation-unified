# Структура объекта транзакции сертификата

## Пример ответа

```json 
{
  "id": 333746,
  "amount": -300,
  "event": "order",
  "order_id": "773",
  "identifier": "",
  "product_id": "17"
}
```

## Описание полей

| Параметр   | Описание                                                                             |
|------------|--------------------------------------------------------------------------------------|
| id         | Идентификатор транзакции                                                             |
| amount     | Сумма транзакции. Отрицательное число означает списание, положительное - компенсация |
| event      | Событие, за которое было списание или компенсация                                    |
| order_id   | Если `event=order`, то тут будет идентификатор заказа                                |
| identifier | Идентификатор участника ПЛ (покупателя)                                              |
| product_id | Идентификатор товара, за который было списание или компенсация, если `event=order`   |
