# Интеграция приложения программы лояльности в кассу Эватор

Наиболее простой способ интеграции программы лояльности в кассу - это покупка и установка приложения в экосистеме Эватор.

Для этого нужно найти приложение в их магазине: оно называется **REES46. Программа лояльности**.

Данное приложение позволяет применять инструменты программы лояльности (скидки, промокоды, сертификаты) непосредственно на кассе Эватор в момент оплаты.

## Установка и первичная настройка 

Дальнейшие шаги подразумевают наличие рабочей экосистемы Эватор. По всем вопросам, касающимся подготовки кассы к работе, можно обратиться к [официальной документации](https://evotor.ru/manuals/).

У вас есть готовая к работе касса. Для интеграции программы лояльности REES46 нужно будет выполнить следующие действия:

1. Найти приложение **REES46. Программа лояльности** в официальном магазине Эватор и приобрести его
2. В вашем личном кабинете Эватор будет представлен список приложений с чекбоксами напротив названий каждого из них. Нажатие на чекбокс запустит процесс установки приложения. После окончания установки программа лояльности на кассе будет готова к работе примерно через пятнадцать минут.
3. Теперь нужно настроить приложение. Сразу после запуска, появится окно первичной настройки, где нужно будет ввести **shop_id** и **shop_secret**. Эти данные можно найти в **личном кабинете REES46**. Настройка производится один раз, вводить данные повторно не потребуется. 

## Сценарий работы программы лояльности в кассе 

Расчет скидок, списание и начисление бонусов, а также иных бенефитов производится на стороне REES46. 

Задача приложения идентифицировать участника программы лояльности или бенефит (например, сертификат), передать данные на сторону платформы, где и будут производиться расчёты, получить данные и применить их к чеку на этапе оплаты. 

На практике это выглядит так:

1. Кассир открывает приложение программы лояльности
2. Вводятся данные клиента (номер телефона - основной идентификатор программы лояльности) для идентификации, либо создания нового участника программы лояльности. Но можно продолжить работу и анонимно
3. Вводятся данные инструментов программы лояльности: промокоды, сертификаты 
4. Приложение отправляет данные платформе REES46, а затем получает результаты расчётов и выводит их на экран
5. Кассир применяет результаты расчётов (скидка) к чеку. Кассовый терминал возвращается к стандартному экрану оплаты
6. Оплата проходит через стандартный функционал кассы

## Отмена операций

Важно учитывать, что корректная отмена операций может происходить только до момента оплаты. 

Кассы Эватор работают в рамках сессий и не хранят информацию о заказах в течение длительного периода. Это значит, что возможности связать исходный заказ и возврат нет. 

Следовательно, автоматической отмены начисления бонусных баллов не будет.
