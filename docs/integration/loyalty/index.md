# Программа лояльности

Раздел описывает методы для работы с программой лояльности.

## Общая терминология

* **Идентификатор участника программы лояльности (ПЛ)** – номер телефона клиента, с которым он регистрируется в программе лояльности в формате `7XXXXXXXXXX` (без скобок, дефисов, пробелов и плюсов, начинается с кода страны).
