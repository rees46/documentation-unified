# Участники программы лояльности

Большинство возможностей программы лояльности требуют от клиента предварительной регистрации в ПЛ. Например, работа с бонусами, стикеры (они же фишки) или акции с таргетингом по участникам с историей покупок.

Поэтому, предварительно нужно настроить регистрацию участника в программе лояльности.

Идентификатором участника программы лояльности является его номер телефона в формате `7XXXXXXXXXX` (для других стран поддерживаются локальные коды). Во всех запросах, кроме регистрации в ПЛ, в поле `identifier` должен быть номер телефона участника без пробелов, скобок, дефисов и других символов. Только цифры.
