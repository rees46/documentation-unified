# Выход из программы лояльности

Клиент может перестать быть участником программы лояльности. После выхода, клиент потеряет свой уровень участника ПЛ. 
Но бонусы, стикеры и история заказов останутся. Поэтому, если клиент заново присоединится, вся его история будет на месте. 
Единственное ограничение, что уровни ПЛ будут подсчитаны на следующий день, а на текущую дату клиент будет находиться на базовом уровне ПЛ. 

Удаление делается следующим запросом:

```
DELETE https://api.rees46.ru/loyalty/members/destroy
```

## Параметры

| Параметр    | Обязателен? | Описание                    |
|-------------|-------------|-----------------------------|
| shop_id     | Да          | API-ключ                    |
| shop_secret | Да          | Секретный ключ API          |
| phone       | Да          | Идентификатор участника ПЛ. |


## Запрос

Пример запроса:

```shell 
curl --header "Content-Type: application/json" \
  --request DELETE \
  --data-binary "@payload.json" \
  https://api.rees46.ru/loyalty/members/destroy
```

Пример JSON-тела:

```json payload.json
{
    "shop_id": "...",
    "shop_secret": "...",
    "identifier": "...",
}
```

## Ответ

Пример ответа сервера:

```json 
{
  "success": true,
  "payload": null
}
```
