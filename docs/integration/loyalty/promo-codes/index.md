# Промокоды

Данный раздел описывает работу промокодов и купонов.

Купон – это тот же промокод, разница лишь в способе получения. Купон выдается покупателю после совершения покупки. А промокод выдается до покупки, например чтобы получить скидку на первый заказ. Но в целом механика работы промокодов и купонов идентична.

Использование и выдача промокодов выполняется автоматически [в процессе покупки](../checkout/index.md).
