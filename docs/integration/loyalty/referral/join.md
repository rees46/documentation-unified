# Регистрация в реферальной программе

Присоединение участника программы лояльности к реферальной программе. Предварительно клиент должен стать участником программы лояльности и только после этого он может стать реферальным партнером.

```
POST https://api.rees46.ru/loyalty/referral/join
```


## Параметры

| Параметр    | Обязателен? | Описание                                           |
|-------------|-------------|----------------------------------------------------|
| shop_id     | Да          | API-ключ                                           |
| shop_secret | Да          | Секретный ключ API                                 |
| identifier  | Да          | Номер телефона клиента, идентификатор участника ПЛ |
| id          | Да          | Идентификатор реферальной программы               |

## Запрос

Пример запроса:

```shell 
curl --header "Content-Type: application/json" \
  --request POST \
  --data-binary "@payload.json" \
  https://api.rees46.ru/loyalty/referral/join
```

Пример JSON-тела:

```json payload.json
{
    "shop_id": "...",
    "shop_secret": "...",
    "identifier": "...",
    "id": 339
}
```

## Ответ

Пример ответа сервера:

```json 
{
  "success": true,
  "payload": {
    "message": "Success",
    "code": "..."
  }
}
```

Расшифровка ответа:

| Параметр        | Описание                                  |
|-----------------|-------------------------------------------|
| success         | Запрос выполнен успешно или нет           |
| payload.message | Сообщение с результатом обработки запроса |
| payload.code    | Промокод реферального партнера            |

В случае ошибки ответ будет в формате:

```json 
{
  "success": false,
  "payload": {
    "message": "Identifier is not valid"
  }
}
```
