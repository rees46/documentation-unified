# История транзакций стикеров

Возвращает список всех транзакций стикеров клиента.

Учитывайте, что стикеры выдаются и применяются по одному в транзакции. Т.е. если клиент использовал 5 стикеров, в базе данных будет 5 транзакций. Каждая транзакция означает один стикер.

Именно поэтому в настройках акций не рекомендуется делать слишком дешевые стикеры (например 1 рубль = 1 стикер). Лучше выдавать их за каждые 500 или 1000 рублей.

```
GET https://api.rees46.ru/loyalty/stickers/history
```

## Параметры

| Параметр    | Обязателен? | Описание                                                        |
|-------------|-------------|-----------------------------------------------------------------|
| shop_id     | Да          | API-ключ                                                        |
| shop_secret | Да          | Секретный ключ API                                              |
| identifier  | Да          | Идентификатор участника программы лояльности                    |
| offset      | Нет         | Смещение от первой транзакции в ответе для постраничного вывода |
| limit       | Нет         | Количество транзакций в ответе. Не более 50                     |

## Запрос

Пример запроса:

```shell 
curl https://api.rees46.ru/loyalty/stickers/history?shop_id=...&shop_secret=...&identifier=...&offset=0&limit=50
```

## Ответ

Пример ответа сервера:

```json 
{
  "success": true,
  "payload": {
    "transactions": [
      {
        "campaign_id": 113,
        "campaign_code": "...",
        "campaign_name": "...",
        "campaign_status": "...",
        "used_in_order": "...",
        "issued_by_order": "...",
        "created_at": "2025-01-14 13:00:12"
      },
      {
        "campaign_id": 2,
        "campaign_code": "...",
        "campaign_name": "...",
        "campaign_status": "...",
        "used_in_order": "...",
        "issued_by_order": "...",
        "created_at": "2025-01-13 13:00:12"
      }
    ]
  }
}
```

Расшифровка ответа:

| Параметр                           | Описание                                                                  |
|------------------------------------|---------------------------------------------------------------------------|
| success                            | Запрос выполнен успешно или нет                                           |
| payload.transactions               | Список объектов транзакций                                                |
| payload.stickers[]                 | Объект транзакции                                                         |
| payload.stickers[].campaign_id     | Идентификатор стикерной кампании                                          |
| payload.stickers[].campaign_code   | Уникальный код кампании                                                   |
| payload.stickers[].campaign_name   | Название кампании                                                         |
| payload.stickers[].campaign_status | Статус кампании                                                           |
| payload.stickers[].used_in_order   | Идентификатор заказа, в котором был использован стикер                    |
| payload.stickers[].issued_by_order | Идентификатор заказа, в котором был выдан стикер в качестве вознагражения |
| payload.stickers[].created_at      | Дата и время, в которое была создана транзакция                           |

В случае ошибки ответ будет в формате:

```json 
{
  "success": false,
  "payload": {
    "message": "Shop not found"
  }
}
```
