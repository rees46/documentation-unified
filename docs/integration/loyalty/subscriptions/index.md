# Подписки и тарифные планы

Сервис "Подписки и тарифы" позволяет создавать аналог подписочных сервисов для управления бенефитами клиентов. Дополнительно интеграция с сервисом "Акции" программы лояльности позволит вам расширить возможности подписок.

Для использования подписок пользователь не обязательно должен быть участником программы лояльности, но участие в ПЛ позволит точнее таргетировать акции, а также использовать бонусную систему.
