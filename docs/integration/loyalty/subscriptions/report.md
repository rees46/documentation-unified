# Отчет для клиента по его подписке

Возвращает информацию о том, сколько клиент сэкономил благодаря своей подписке.

```
GET https://api.rees46.ru/loyalty/subscriptions/members/report
```

## Параметры

| Параметр     | Обязателен? | Описание                                                        |
|--------------|-------------|-----------------------------------------------------------------|
| shop_id      | Да          | API-ключ                                                        |
| shop_secret  | Да          | Секретный ключ API                                              |
| identifier   | Да          | Номер телефона клиента, идентификатор участника ПЛ              |
| subscription | Да          | Уникальный код подписки                                         |
| depth        | Да          | На сколько месяцев назад собирать данные для отчета. От 0 до 12 |

Если параметр `depth` не указан или равен `0`, то отчет будет содержать только один текущий месяц. Если `1`, то в отчет попадут текущий и предыдущий месяцы. Если `4`, то текущий и 4 предыдущих месяца. И так далее.

## Запрос

Пример запроса:

```shell 
curl https://api.rees46.ru/loyalty/subscriptions/members/report?shop_id=...&shop_secret=...&identifier=...&subscription=...&depth=5
```

## Ответ

Пример ответа сервера:

```json 
{
  "success": true,
  "payload": {
    "message": "Successful request",
    "subscription": {
      "id": 1,
      "code": "premium",
      "name": "Премиум-подписка"
    },
    "report": [
      {
        "month": "2025-12-01",
        "orders": 33,
        "saved": 371881
      },
      {
        "month": "2026-01-01",
        "orders": 12,
        "saved": 17381
      },
      {
        "month": "2026-02-01",
        "orders": 7,
        "saved": 695484
      },
      ...
    ]
  }
}
```

В случае ошибки ответ будет в формате:

```json 
{
  "success": false,
  "payload": {
    "message": "API secret is not correct"
  }
}
```

Расшифровка ответа:

| Параметр                           | Описание                                                                                                                                                                               |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| success                            | Запрос выполнен успешно или нет                                                                                                                                                        |
| payload.message                    | Сообщение с результатом обработки запроса                                                                                                                                              |
| payload.subscription               | Объект с информацией о подписке                                                                                                                                                        |
| payload.subscription.id            | ID подписки                                                                                                                                                                            |
| payload.subscription.code          | Уникальный код подписки                                                                                                                                                                |
| payload.subscription.name          | Название подписки                                                                                                                                                                      |
| payload.report[]                   | Массив со списком помесячных отчетов                                                                                                                                                   |
| payload.report[].month             | Месяц, за который собран отчет. Всегда возвращается первое число месяца, но отчет собран за весь месяц                                                                                 |
| payload.report[].orders            | Количество не отмененных заказов с участием подписки клиента                                                                                                                           |
| payload.report[].saved             | Сумма денег, сэкономленных по акциям, доступным только по подписке                                                                                                                     |
