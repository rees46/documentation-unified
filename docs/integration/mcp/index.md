# MCP

`MCP` (`Model Context Protocol`) сервер предоставляет доступ к данным REES46 в режиме чтения. Через него можно задавать вопросы и ставить задачи `LLM` на естественном языке, а модель будет последовательно использовать доступные инструменты и ресурсы для получения результата.

Это удобно для быстрых проверок, разовых аналитических запросов и выгрузки данных без ручной работы с интерфейсами и API.

> [!IMPORTANT]
> MCP сервер находится в активной разработке. Список доступных инструментов и ресурсов может меняться.

## Как это работает

После подключения MCP сервера в совместимом клиенте вы можете обращаться к данным REES46 прямо в чате. Модель сама выбирает, какие инструменты использовать, чтобы ответить на вопрос, собрать выборку или подготовить файл для экспорта.

## Что можно делать через MCP

### Быстрый поиск и анализ

Примеры запросов:

> Покажи доступные мне магазины

> Покажи последние события пользователя `mail@example.org`

> Выведи список товаров бренда `Apple` и отсортируй их по цене

### Подготовка отчётов

MCP сервер также может формировать большие отчёты для скачивания в формате `CSV`. Если нужен именно файл, а не краткий ответ в чате, явно укажите это в запросе, например: `экспортируй`.

Примеры:

> Экспортируй список заказов за март 2026 года в CSV

> Подготовь CSV с товарами бренда `Apple`, которые есть в наличии

## Рекомендации по формулировке запросов

Чем точнее сформулирован запрос, тем полезнее будет результат. По возможности указывайте:

- период, например: `за последние 30 дней`;
- сущность, с которой нужно работать: `заказы`, `клиенты`, `товары`, `события`;
- фильтры: email, бренд, сегмент, статус, диапазон цен;
- желаемый формат ответа: краткая сводка в чате или `CSV`-экспорт.

## Ограничения

- MCP сервер предоставляет доступ только в режиме чтения.
- Набор доступных инструментов зависит от текущей версии сервера и может изменяться.
- Скорость ответа зависит от объёма данных и сложности запроса.
- Для больших выборок лучше сразу запрашивать экспорт в `CSV`, а не ответ в чате.
- Файлы с отчетами доступны ограниченное время, обычно не больше 1 часа
