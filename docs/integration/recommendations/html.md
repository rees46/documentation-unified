# Подключение товарных рекомендаций на сайте через DIV

Самый простой способ установки товарных рекомендаций на сайт – размещение DIV-блоков.

JS SDK автоматически находит все DIV-блоки с определенным классом, получает из них идентификаторы блоков, запрашивает информацию и верстку по ним и вставляет готовые товарные подборки на сайт.

Для автоматической отрисовки достаточно установить DIV-блоки на странице в формате:

```html
<div
    class="rees46-recommend"
    data-recommender-block="dynamic"
    data-recommender-code="..."
    data-recommender-item="..."
    data-recommender-exclude="..."
    data-recommender-category="..."
    data-recommender-categories="..."
    data-recommender-locations="..."
    data-recommender-search_query="..."
    data-recommender-brands="..."
    data-recommender-callback="..."
    data-recommender-error="..."
    data-recommender-min="..."
    data-recommender-collapse="..."
/>
```

Описание атрибутов тега:

| Параметр                      | Тип      | Обязателен? | Описание                                                                                                                                                                                              |
|-------------------------------|----------|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| class                         | String   | Да          | Должен содержать класс `rees46-recommend`. Для white label кластеров `rees46` меняется на код, соответствующий white label                                                                            |
| data-recommender-block        | String   | Да          | Тип блока. Всегда `dynamic`                                                                                                                                                                           |
| data-recommender-code         | String   | Да          | Уникальный код блока из личного кабинета                                                                                                                                                              |
| data-recommender-item         | String   | Нет         | Идентификатор товара, на странице которого находится блок. Обязателен для алгоритмов "Похожие" и "С этим товаром покупают"                                                                            |
| data-recommender-exclude      | String   | Нет         | Список идентификаторов продуктов (через запятую), которые должны быть исключены из рекомендаций                                                                                                       |
| data-recommender-category     | String   | Нет         | Идентификатор категории для алгоритма "Популярные в категории"                                                                                                                                        |
| data-recommender-categories   | String   | Нет         | Идентификаторы категорий (через запятую) для алгоритма "Популярные в категории", если нужно передать больше одной категории                                                                           |
| data-recommender-locations    | String   | Нет         | Идентификатор текущей локации (города), для которого вернуть товарные рекомендации с учетом наличия товаров в этом городе                                                                             |
| data-recommender-search_query | String   | Нет         | Поисковый запрос со страницы полного поиска для алгоритма "Люди, искавшие это, купили эти товары"                                                                                                     |
| data-recommender-brands       | String   | Нет         | Список брендов через запятую. Будут возвращены только товары брендов, перечисленных в этом параметре                                                                                                  |
| data-recommender-min          | Integer  | Нет         | Минимальное количество товаров в ответе API, при котором отобразить блок. Если API вернул меньше товаров, чем это значение, блок не будет отображен                                                   |
| data-recommender-collapse     | Boolean  | Нет         | Позволяет отключить группировку товаров по свойству товара `group_id`. Значение `true` (по умолчанию) даёт только один товар из группы                                                                |
| data-recommender-callback     | String   | Нет         | Имя JS функции, куда уйдет ответ сервера, если вы хотите обработать его сами. Если не указать, то JS SDK автоматически обработает ответ и сгенерирует верстку из шаблона, заданного в личном кабинете |
| data-recommender-error        | String   | Нет         | Имя JS функции, куда уйдет ответ сервера в случае ошибочного запроса                                                                                                                                  |
