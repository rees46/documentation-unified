# Товарные рекомендации

Персональные товарные рекомендации – это сервис, который анализирует поведение посетителя и рекомендует ему небольшое количество товаров (от 6 до 20) на основе заданных стратегий, алгоритмов, фильтров и пр.

Управление правилами товарных рекомендаций выполняется в личном кабинете. На фронте вам нужно только отобразить то, что вернул API.

Есть два способа подключения:

1. Простой способ с помощью DIV-тега на сайте.
2. Ручной способ с помощью SDK и REST API.
