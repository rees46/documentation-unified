# Полный поиск

Полный поиск выполняется на странице, специально выделенной под полный поиск. Пользователь сюда попадает из окна [быстрого поиска](../instant/index.md) при клике на ссылку "Показать все" или нажатии на Enter после ввода поискового запроса.

Страница полного поиска состоит из:

- [результатов](./api.md) полного поиска;
- [сайдбара](./facet.md) с фильтрами по параметрам (фасетный поиск).
