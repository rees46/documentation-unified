# Поиск товаров

Поиск товаров состоит из двух независимых компонентов:

1. [Быстрый поиск](./instant/index.md): текстовое поле в шапке сайта, при постепенном вводе которого поиск находит подходящие товары и выводит их в дропдауне.
2. [Полный поиск](./full/index.md): отдельная страница с результатами поиска и сайдбаром с фасетами (параметрами) для уточнения поисковых запросов по параметрам товаров.
