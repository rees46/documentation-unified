# Виджет быстрого поиска на сайте

Чтобы активировать быстрый поиск, достаточно просто в текстовое поле поиска добавить класс `rees46-instant-search`. JS SDK автоматически найдет этот тег и активирует на нем быстрый поиск.

:::info White-label
Для white label вместо `rees46` нужно указать код white label проекта. 
:::

```html
<input type="text" name="suggest" class="rees46-instant-search" />
```

На этом подключение быстрого поиска на сайт завершено.
