# Виджет быстрого поиска

Цель виджета быстрого поиска: пока клиент вводит то, что ему нужно найти, поиск по каждой новой букве уточняет поисковый запрос и пытается предположить, что именно ищет клиент.

Виджет на сайте можно установить быстро, просто добавив `CSS`-класс к текстовому полю. Либо реализовать собственный виджет, используя платформу только как источник данных.
