# Баннеры на сайте

По умолчанию JS SDK автоматически ищет `DIV` блоки со специальными атрибутами и конвертирует их в слайдеры. Также JS SDK автоматически отслеживает просмотры и переходы, добавляя события в CDP для последующей аналитики.

Достаточно установить `DIV` в нужных местах страницы сайта и JS SDK сделает всё остальное:

```html 
...
<div class="rees46-slider" data-slider-code="SLIDER_CODE"></div>
...
```

Уникальный код слайдера берется со страницы настроек слайдера в панели управления.

## Ручная обработка блоков

Если вас не устраивают служебные классы REES46 в HTML, вы можете самостоятельно запросить баннеры слайдера и обработать ответ:

```javascript 
r46('slider', { code: "SLIDER_CODE" }, r => console.log(r))
```

Но в этом случае вам придется самостоятельно создавать карусель баннеров, трекинг открытий и кликов и пр. 

Если же вы хотите вызвать отрисовку конкретного слайдера в нужном вам DIV-блоке, то можно вызвать JS SDK так:

```javascript 
r46('slider', 
  { 
    code: 'demoshop_main', 
    block: 'ID тега HTML, внутри которого необходимо установить карусель',
    config: {
      dotBgColor: 'red', 
      dotActiveBgColor: 'green', 
      dotHoverBgColor: 'yellow'
    }
  }
)
```
