# Интеграция модуля Stories

Сервис "Сторис" работает на сайте, а также в мобильных приложениях. 

Интеграция максимально простая: SDK автоматически отображает иконки кампаний, в SDK встроен плеер сторис, а также автоматически обрабатываются действия пользователей и сохраняются в CDP для последующей аналитики.

Нет никакой необходимости создавать свой собственный плеер сторис.
