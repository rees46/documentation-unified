# Сторис на сайте

Вы можете использовать разные блоки для вставки сторис на разных страницах сайта: один на главной, один на экране распродажи, в корзине, в личном кабинете, в определенной категории товаров. У вас нет никаких ограничений.

## Стандартный режим

JS SDK имеет встроенный плеер сторис и работает автоматически. Для установки сторис на странице вам нужно просто добавить `DIV` в нужном месте страницы:

```html
<div class="rees46-stories" data-code="STORY BLOCK CODE" />
```

Где вместо `STORY BLOCK CODE` нужно указать уникальный код блока сторис из личного кабинета.

Вся дальнейшая работа по рендеру блока, обработке нажатий и отображению слайдов выполняется SDK автоматически.

## GTM и SPA

`DIV` должен быть вставлен на страницу до запуска инициализации JS SDK. Если вы используете GTM или другие скрипты, которые вставляют блоки после загрузки страницы, нужно вручную инициализировать сторис такой командой:

```js 
r46('stories_init')
```

То же самое нужно делать в случае навигации по `single page application` (`SPA`) сайтам.
