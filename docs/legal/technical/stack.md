# Технический стек

## Базы данных

- PostgreSQL: для изменяемых данных, клиентские данные, кампании и другие сущности платформы;
- Clickhouse: для аналитических и поведенческих данных;
- ElasticSearch: для древовидных данных и товарных каталогов.

## Языки программирования

- Ruby: личный кабинет и некоторые API
- PHP: высоконагруженные API и сервисы
- Go: высоконагруженные сервисы
- Python: ML-сервисы и аналитические сервисы без высокой нагрузки
- Rust: высоконагруженные ML-сервисы

## Операционные системы и инфраструктура

Управление инфраструктурой: Kubernetes.

Операционные системы:

- Talos (для K8S)
- Ubuntu 24
