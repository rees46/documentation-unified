# Ссылка на отписку в кнопках действий WhatsApp рассылок

По умолчанию клиент может отписаться только от email рассылок с помощью ссылки на отписку в один клик в подвале письма. 

В мессенджерах маркетологи обычно не увеличивают текст сообщений, чтобы не отвлекать клиента от основного содержания сообщения. Поэтому не вставляют ссылку на отписку и клиент может отписаться только с помощью блокировки контакта.

WhatsApp поддерживает кнопки действий, и в одну из них можно добавить действие с отпиской. Действие представляет собой просто переход по ссылке с определенным адресом. 

:::info Защита от автоматической отписки
WhatsApp не делает превью ссылки под кнопкой, поэтому не будет открывать ссылку автоматически на своей стороне и тем самым не отпишет подписчика.
:::

## Как добавить кнопку

1. В настройках кампании добавьте дополнительную кнопку.
2. Выберите тип кнопки `URL`.
3. В поле `Надпись на кнопке` введите текст кнопки, например `Отписаться`.
4. Вставьте текст <code v-pre>https://api.rees46.ru/opt-out/{{message_code}}</code> в значение поля `Действие при нажатии на кнопку`.  

При отправке сообщения сервис автоматически заменит переменную <code v-pre>{{message_code}}</code> на уникальный код сообщения, и подписчик, перейдя по ссылке, отпишется от данного типа рассылки.
